INTRODUCTION
------------

This script will generate a random song structure with random associated phrasing for each section.


REQUIREMENTS
------------

This module requires the following external modules:

 * random (documentation: https://docs.python.org/3/library/random.html)
 * pandas (documentation: https://pandas.pydata.org/docs/)


CONFIGURATION
-------------

 * random-song-structure-generator.py - script
 * songStructureVariables.py          - section variables for randomization


INSTRUCTIONS
-------------

Command:
    $ python3 random-song-structure-generator.py
Output: 
    random-song-structure.csv
